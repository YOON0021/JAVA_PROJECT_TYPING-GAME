프로젝트명: ZOMBIE DEFENSE SURVIVAL

1. 작품 개요
“움직이는 단어 타이핑 게임“에서 조금 추가하여 ”좀비 서바이벌 게임”을 개발해 보았습니다.
게임메인화면에서는 난이도 조절과 단어 추가, 랭킹 보기등의 기능을 추가하여 게임자체의 완성도를 높였습니다.
게임화면에는 한쪽에는 플레이어의 기지가 있고 다른 쪽에서는 좀비들이 몰려옵니다. 좀비들은 각각 독립된 스레드로 움직이며, 동시에 여러 개가 다른 크기와 속도로 화면에 등장하여 플레이어의 기지 쪽으로 몰려옵니다.
또한, 좀비들은 여러 개의 이미지(gif)를 통해 생동감 있게 움직이도록 설정했습니다. 사용자는 화면의 입력창에서 단어를 정확하게 입력하여 좀비를 제거해야 하며 입력에 성공하면 좀비가 이펙트와 효과음과 함께 제거되고 점수가 증가합니다.
좀비를 제거하지 못해 플레이어의 목숨이 다 사라지거나 일정 점수를 획득해 게임이 끝나게 되면 랭킹화면에 이름을 입력할 수 있도록 설정했습니다.
